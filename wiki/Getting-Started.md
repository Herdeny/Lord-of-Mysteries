# 入门与占卜家序列 9–7

> 当前版本：0.8.5-1.20.1
>
> 设计基线：v0.8 · M1 验收 / M2 预研

首次登录会获得《神秘学手账》和调查员罗盘。右键手账查看下一步，
按 `N` 打开非凡者档案。

## 首十分钟

1. 输入 `/pm mistcity` 前往雾都前哨，右键委托板领取“走失的煤球”。
2. 输入 `/pm camp` 前往新手营地，找到黑猫后返回委托板结算。
3. 搜索营地材料，使用 `/pm guide` 重读目标，使用 `/pm rules` 理解风险。
4. 制作坩埚；调查员罗盘也可由指南针、纸和紫水晶重新合成。

## 普通人 → 占卜家

1. 探索调查员营地或制作坩埚。
2. 依次加入灵性草药、占卜水晶；月华水提高品质。
3. 保持温度接近 70，取出并服用占卜家魔药。
4. `V` 使用灵视，`B` 进行简易占卜。

## 占卜家 → 小丑

- 消化必须达到 100%。
- 材料：深灰灵体之泪、石楠、灵性酒精。
- 理想温度：65。
- `R` 使用纸牌飞刃；潜行右键村民使用表情操控。
- 被动降低摔落和击退，并提供 15% 近战闪避。

## 小丑 → 魔术师

- 消化必须达到 100%。
- 材料：幻形蛇毒腺、灰烬粉、银屑。
- 理想温度：90。
- 幻形蛇在主世界低概率生成，靠近前保持隐匿；灵视提灯可强制显形。
- `Z` 火焰跳跃、`X` 替身纸人、`C` 空气子弹、`M` 舞台幻术。

## 风险

重复扮演会衰减；污染和失控压力会累积。占卜家失控体会泄露坐标并预判攻击。

- 镇静熏香：压力 -18、污染 +2，只能在压力大于 0 时使用。
- 灵视提灯：显形 16 格内敌对生物并强制幻形蛇显形 10 秒，消耗 3 灵性；无法支付时压力 +2。
- 完整睡眠：每个游戏日首次整夜睡眠降低 20 压力；提前离床不触发。
- 精神守护符：自动抵消一次重度精神事件，不能阻止污染 100 的完整失控。
- 灵体微光：飞行并周期抽取灵性；永燃火柴盒对其造成双倍伤害。
- 灰烬傀儡：攻击施加虚弱与缓慢；水、雨和气泡会持续侵蚀它。
- 幻形蛇：接近前隐形，毒腺是魔术师魔药主材料。

## M2 预研入口

调查营地可能出现配方残页。右键残页会依次解锁两条途径传闻、调查委托和雾都镇区知识；
任意传闻都会开放 `/pm handbook 7`。空白手稿、神秘墨水与雾都晨报已可制作，
偷盗者与学徒序列 9 已进入可玩预研：偷盗者使用 `Y` 妙手空空、潜行触发潜影步、`U` 应急脱身；
学徒使用 `V` 灵性视野、`Y` 小空间戏法、`U` 制作灵墨知识副本。

观众和猎人现已补齐序列 7：心理医生使用 `Y` 心理安抚、`U` 心灵震慑和潜行心理隐身；
纵火家使用 `Y` 火焰长枪、`U` 焰环和火焰亲和。详见
[心理医生与纵火家序列 7](M2-Sequence-7-Psychiatrist-and-Pyromaniac)。

偷盗者/学徒现已补齐序列 8-7：诈骗师/解密学者使用换位、假身、符文、开锁和抹痕；
戏法大师/占星人使用空间转移、知识链接、镜门、星象预兆和团队庇护。详见
[偷盗者与学徒序列 8-7](M2-Sequence-8-7-Thief-and-Apprentice)。

雾都前哨、寻猫委托和“失踪调查小队·第一阶段”现已可玩；详情见
