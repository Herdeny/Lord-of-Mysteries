# 入门与占卜家序列 9–7

> 当前版本：0.4.0-1.20.1
>
> 设计基线：v0.6 · M1

首次登录会获得《调查员残缺手稿》和调查员罗盘。右键手稿查看下一步，右键罗盘寻找营地，
按 `N` 打开非凡者档案。

## 首十分钟

1. 右键罗盘，记录方向、距离和坐标；丢失时可输入 `/pm camp`。
2. 前往出生点外约 10–16 区块的新手营地，打开箱子收集基础材料。
3. 使用 `/pm guide` 重读目标，使用 `/pm rules` 理解灵性、污染和压力。
4. 制作坩埚；调查员罗盘也可由指南针、纸和紫水晶重新合成。

## 普通人 → 占卜家

1. 探索调查员营地或制作坩埚。
2. 依次加入灵性草药、占卜水晶；月华水提高品质。
3. 保持温度接近 70，取出并服用占卜家魔药。
4. `V` 使用灵视，`B` 进行简易占卜。

## 占卜家 → 小丑

- 消化必须达到 100%。
- 材料：深灰灵体之泪、石楠、灵性酒精。
- 理想温度：65。
- `R` 使用纸牌飞刃；潜行右键村民使用表情操控。
- 被动降低摔落和击退，并提供 15% 近战闪避。

## 小丑 → 魔术师

- 消化必须达到 100%。
- 材料：幻形蛇毒腺、灰烬粉、银屑。
- 理想温度：90。
- 幻形蛇在主世界低概率生成，靠近前保持隐匿。
- `Z` 火焰跳跃、`X` 替身纸人、`C` 空气子弹、`M` 舞台幻术。

## 风险

重复扮演会衰减；污染和失控压力会累积。占卜家失控体会泄露坐标并预判攻击。

- 镇静熏香：压力 -18、污染 +2，只能在压力大于 0 时使用。
- 灵视提灯：显形 16 格内敌对生物 10 秒，消耗 3 灵性；无法支付时压力 +2。
- 灵体微光：飞行并周期抽取灵性，掉落灵性盐。
- 灰烬傀儡：攻击施加虚弱与缓慢，掉落灰烬丝线。
- 幻形蛇：接近前隐形，毒腺是魔术师魔药主材料。
