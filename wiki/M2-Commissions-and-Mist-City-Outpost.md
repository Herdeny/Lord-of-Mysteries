# 雾都委托与失踪调查小队第一阶段

> 版本：0.9.9-1.20.1 · 设计基线：v0.9 · M2 调查与生活

当前已实现“轻量前哨与调查链第一阶段可玩”，不代表完整维多利亚镇区或设计文档中的
6–10 小时精制链一已经全部完成。

## 如何开始

1. 输入 `/pm mistcity` 前往世界种子确定的雾都前哨。
2. 右键调查板比较案件摘要、报酬和锁定条件，再主动接取；切换“证据档案”查看缺失记录。
3. 在界面或 `/pm commission status` 查看步骤；使用 `/pm camp` 或 `/pm case` 定位地点。
4. 救出记者、护送回前哨并完成三波夜袭后，与值夜者联络员结算。

## 第一单：走失的煤球

- 无需成为非凡者。
- 使用 `/pm camp` 前往调查营地。
- 找到持久化黑猫“煤球”并靠近。
- 返回前哨结算 24 便士和 3 点事务所声望。

旧存档若没有黑猫，任务会在营地附近补生成，不要求重新开图。

## 失踪的调查小队·第一阶段

| 步骤 | 目标 |
|---|---|
| 1–3 | 接取报社委托、确认出发记录、抵达调查营地 |
| 4–6 | 取得染血笔记、达到序列 9、击败三只神秘生物 |
| 7 | 与值夜者联络员交谈，取得继续调查许可 |
| 8 | 使用 `/pm case` 搜查废弃教堂 |
| 9 | 抵达邪教救援营地 |
| 10 | 选择强攻、潜入或占卜路线，再右键幸存记者艾琳完成营救 |
| 11 | 护送艾琳返回雾都前哨 |
| 12 | 夜间在前哨击退三波追兵 |
| 13 | 与值夜者联络员交谈并结算 |

奖励为 144 便士、10 点事务所声望和“烧焦的名单”。名单只开启链二线索，链二目前不可接取。

## 地点与 NPC

- 雾都前哨：出生区外 3–5 区块，包含委托板、报社办事员和值夜者联络员。
- 废弃教堂：出生区外 6–8 区块，包含染血笔记、配方残页和调查补给。
- 邪教救援营地：出生区外 10–13 区块，包含囚区、材料箱和幸存记者。
- 神秘学家小屋：出生区外 8–10 区块，包含鉴定师、试剂台和真假配方案件。
- 四处结构都由世界种子确定，并通过 SavedData 支持旧存档补生成。
- 任务 NPC 复用原版村民渲染，但右键不会打开交易界面，也不会被玩家误杀。

## 委托命令

| 命令 | 用途 |
|---|---|
| `/pm mistcity` | 定位雾都前哨 |
| `/pm city` | 查看报社、事务所和巡警岗亭服务目录 |
| `/pm case` | 定位废弃教堂与邪教救援营地 |
| `/pm commission board` | 在调查板附近打开可视化案件界面 |
| `/pm commission list` | 可用案件与完成状态 |
| `/pm commission accept <id>` | 接取指定案件 |
| `/pm commission status` | 当前步骤、波次与账本 |
| `/pm commission abandon` | 放弃当前案件 |
| `/pm commission approach assault|stealth|divination` | 锁定记者救援路线 |
| `/pm formula inspect|verdict` | 鉴定并提交密封配方真伪 |
| `/pm party` | 查看队伍持久调查账本 |
| `/pm party sync` | 中途加入案件或追赶离线进度 |

## 数据与存档

- `commissions/` 定义委托、至少两种解法、报酬、声望、冷却和任务链引用。
- 调查板以服务端快照显示案件与三案证据档案；客户端只发送接取/放弃意图，不能提交证据。
- `quests/` 定义十三步第一阶段，并使用 v0.8 冻结的 15 类任务目标。
- Capability schema 19 保存并自修复余额、活动案件、步骤、记者 UUID、夜袭、救援路线、城市工作、完成集合、玩家假说、结案复盘、特性账本与扮演 v2 字段。
- 夜袭敌人带队伍隔离键；自然生成怪物和其他调查队的追兵不会被误算为本队波次。
- 同记分板队伍 2–4 名成员共享 `coop.shared_progress`；护送/夜袭只由确定性在线协调者驱动。
- 超过 `max_party` 安全降级为单人；已登记成员离线后上线自动追赶，中途加入使用 `/pm party sync`。
- 每名成员独立结算奖励；换队/退队残留已自动清理，正式队伍 GUI、邀请/队长权限和队名复用管理尚未实现。

## 当前边界

- 完整维多利亚镇区、正式报社/教堂模板和剧情演出尚未实现。
- 前哨已有版本化事务所/巡警功能亭、调查补给交易和安全室恢复，但不等于完整警局、医院或城市经济完成。
- 神秘学家小屋、真假配方鉴定与三解已有轻量可玩实现；正式警戒 AI 和演出尚未实现。
- 账本已用便士记录并显示金镑/苏勒/便士，但实体货币、银行和当铺尚未实现。
- M1 两小时合同、自动连续性与双启动专服门禁已完成；真人体验继续作为发布质量回归。
