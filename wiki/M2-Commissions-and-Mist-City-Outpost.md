# 雾都前哨、委托与失踪调查小队序章

> 版本：0.8.3-1.20.1 · 设计基线：v0.8 · M2 可玩预研

本批把雾都从报纸传闻推进为第一段可玩的社会循环。当前只宣称“轻量前哨与任务序章可玩”，
不代表完整镇区、NPC 或设计文档中的 6–10 小时链一已经完成。

## 如何开始

1. 输入 `/pm mistcity`，前往世界种子确定的雾都前哨。
2. 右键委托板接取推荐案件。
3. 右键调查委托书或输入 `/pm commission status` 查看当前步骤。
4. 完成全部目标后返回同一委托板结算。

## 第一单：走失的煤球

- 无需成为非凡者。
- 使用 `/pm camp` 前往调查营地。
- 找到持久化黑猫“煤球”并靠近。
- 返回前哨结算 24 便士和 3 点事务所声望。

旧存档若没有黑猫，任务会在营地附近补生成，不要求重新开图。

## 失踪的调查小队·序章

| 步骤 | 目标 |
|---|---|
| 1 | 接取报社委托 |
| 2 | 在前哨确认最后出发记录 |
| 3 | 抵达废弃调查营地 |
| 4 | 取得染血调查笔记 |
| 5 | 任一首发途径达到序列 9 |
| 6 | 击败三只特殊神秘生物或失控体 |
| 7 | 返回委托板结算 |

奖励为 144 便士、10 点事务所声望和“烧焦的名单”。名单只开启链二线索，链二目前不可接取。

## 委托命令

| 命令 | 用途 |
|---|---|
| `/pm commission` | 当前案件与账本 |
| `/pm commission list` | 可用案件与完成状态 |
| `/pm commission accept <id>` | 接取 `lost_cat` 或 `missing_investigation_squad` |
| `/pm commission status` | 当前步骤与计数进度 |
| `/pm commission abandon` | 放弃当前案件 |

## 数据与存档

- `commissions/` 定义委托、至少两种解法、报酬、声望、冷却和任务链引用。
- `quests/` 定义有序步骤、引导文本、目标类型、目标 ID 和计数。
- v0.8 冻结 15 类任务目标；未知类型会被 Java 和构建脚本共同拒绝。
- Capability schema 11 保存余额、活动案件、步骤、进度、完成集合和冷却。
- `coop.shared_progress` 已进入 schema，但队伍共享推进尚未实现。

## 当前边界

- 完整维多利亚镇区、报社、教堂、事务所 NPC 尚未实现。
- 废弃教堂、神秘学家小屋、真假配方、记者救援、护送和夜袭防守尚未实现。
- 账本已用便士记录并显示金镑/苏勒/便士，但实体货币、银行和当铺尚未实现。
- M1 一小时、死亡重连和专用服务器验收仍是正式路线门禁。
