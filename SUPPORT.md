# Support

## 获取帮助

1. 阅读 `README.md`、[Wiki](https://github.com/Herdeny/Lord-of-Mysteries/wiki)
   和[资料站](https://herdeny.github.io/Lord-of-Mysteries/)。
2. 搜索已有 Issues，确认问题没有被记录。
3. 使用匹配的 Issue 表单，并提供 Forge、Java、模组版本、最小复现步骤和日志。

请勿发送整个游戏目录。日志应删除用户名、服务器地址、令牌和其他个人信息。

## 支持范围

维护者优先支持仓库 `main` 和最新 GitHub Release。自行修改的 Jar、整合包冲突、
第三方启动器、旧版本 Forge 与未授权转载版本不保证支持。

安全漏洞请遵循 `SECURITY.md`，不要公开披露。知识产权问题使用 `legal/ip`
标签，并先阅读 `NOTICE.md`。
