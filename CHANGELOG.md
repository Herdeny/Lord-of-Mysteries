# Changelog

本项目采用 [Semantic Versioning](https://semver.org/) 的预发布阶段规则，并在版本号后附加
Minecraft 兼容版本。版本与更新时间以 `project-status.json` 为准。

## [0.2.1-1.20.1] - 2026-07-04

### Added

- `roadmap.json` 路线单一来源和自动生成的 `ROADMAP.md`。
- Pages 路线数据、README 路线表和 Wiki 里程碑自动同步。
- v0.6 的 M0-M7 阶段、验收门禁和 MVP / EP1 / EP2 内容规模目标。
- UTF-8 原创化映射表与机器可读的 `docs/ip_mapping.csv`。

### Changed

- 当前阶段按 v0.6 硬门禁从旧 M2 纠正为 M1：占卜家序列 9-7 纵切尚未验收。
- M1、M2 执行计划、代码模块注释、Issue 模板和贡献流程切换到 v0.6 路线。
- 灰雾与塔罗会移动到 M4 / EP1，M3 调整为 MVP 公测内容层。

## [0.2.0-1.20.1] - 2026-07-04

### Added

- v0.6 设计基线审计与仓库实施边界说明。
- 可制作、可放置的仪式粉笔阵纹，以及祭坛半径 3 的八点多方块阵列。
- 仪式材料、环境、阵列完整度和主持者资格评分。
- 完美、成功、失败、严重失败与灾变五档结算。
- 失败时的失控压力、污染、失控体和非破坏性爆炸结果。
- 仪式进行期间每 20 tick 复检阵列；结构破坏时取消并保留材料。
- 仪式结构与结算纯逻辑测试。

### Changed

- 净化封印改用通用仪式底座，并在所有结算后统一消耗可消耗材料。
- README、Pages、Wiki 与 M2 计划同步到设计文档 v0.6 和仪式 M2-3 状态。

## [0.1.0-1.20.1] - 2026-07-04

### Added

- 猎人序列 9 与挑衅者序列 8 的完整晋升、魔药和扮演数据。
- 45 秒战斗追踪、户外荒野感知与自然生成目标标记。
- 可放置、绑定所有者并在触发后消耗的猎人捕兽夹。
- 服务端权威的挑衅、激怒和战斗意志能力，以及 `K` / `L` 默认按键。
- 猎人专属户外灵性恢复加成、中英双语资源和单元测试。

### Changed

- README、GitHub Pages、Wiki 与 M2 计划同步为三条可玩途径。
- 玩家数据 schema 和网络协议升级至版本 3。

## [0.0.1-1.20.1] - 2026-07-03

### Added

- 占卜家序列 9 完整可玩闭环。
- 观众序列 9 与读心者序列 8、二次晋升和精神能力。

- 永燃火柴盒、净化封印、调查员营地和非凡者档案。
- 中英双语、JUnit 测试、Build、CodeQL、Pages 与标准仓库治理文件。

- Pages 部署增加一次受控重试。
