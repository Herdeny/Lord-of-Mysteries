# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| Latest commit on `main` | Yes |
| Latest published release | Yes |
| Older snapshots and forks | No |

## Reporting a vulnerability

请不要为可利用漏洞创建公开 Issue。优先使用仓库 Security 页面的
“Report a vulnerability”入口：

<https://github.com/Herdeny/Lord-of-Mysteries/security/advisories/new>

报告应包含受影响版本、复现步骤、影响、日志以及建议修复方式。不要附带真实
玩家数据、访问令牌或不必要的攻击载荷。

维护者会尽力在 7 天内确认报告，并在 30 天内给出修复或状态更新。时间目标
不是服务等级承诺。修复发布前请避免公开细节。

普通崩溃、模组兼容或玩法问题不属于安全漏洞，请使用 Bug report。
