## 切换策略
- 代码层：`mod_id`、JSON 的 `id` 字段、ResourceLocation 均使用稳定英文代号（如 `seer`），
  不含原作中文专有名词，发布时只需改 `display_name` 与 lang 文件即可完成原创化。
- 展示层：所有玩家可见文本走 `lang/*.json`，发布版替换为原创名即可，无需改逻辑。
