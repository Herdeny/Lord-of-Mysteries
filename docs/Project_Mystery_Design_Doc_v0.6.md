## 1. 项目定位

Project Mystery 是以知识、代价、扮演和调查为核心的 Forge 1.20.1 生存冒险模组。
玩家不是沿技能树升级，而是在信息不完整的世界里寻找配方、承担魔药风险、遵守扮演准则，
并通过调查、仪式、组织和世界事件推进。

### 设计支柱

1. **知识即资源**：正确配方、仪式和禁忌是力量前提，错误知识会产生真实后果。
2. **力量有代价**：能力越强，污染、疯狂和失控风险越高。
3. **行为塑造成长**：魔药通过符合角色准则的行为消化，挂机不产生有效成长。
4. **信息不完全可靠**：占卜、灵视和梦境提供线索，不提供万能答案。
5. **原版生存仍重要**：地形、天气、月相、材料、战斗和探索必须参与神秘学系统。
6. **灰雾是社交系统**：不是传送门或作弊仓库。
7. **多人能力可审计**：读心、偷盗、控制和 PvP 能力必须有开关、保护和日志边界。
8. **特性守恒**：晋升、死亡、失控和非凡生物掉落共同构成材料循环。
9. **恐怖来自留白**：信息污染、声音、规则和后果优先于廉价跳脸。
10. **正常世界必须成立**：经济、报纸、火车、委托和组织构成神秘学之外的社会层。

### 明确不做

- 不做无代价技能树、挂机经验或全配方直接公开。
- 不复刻原作主角团线性剧情；玩家拥有自己的调查故事。
- 序列 3 及以上不作为玩家可达内容，只作为世界事件、NPC 或圣物来源。
- M3 之前不提前完成 EP1 内容；每个里程碑必须通过硬门禁。

## 2. 技术与数据原则

- 玩家状态使用 Forge Capability，当前 schema 为 V4。
- 能力由客户端按键发起，服务端校验途径、序列、冷却、灵性、目标和 PvP 规则。
- 存档 ID、网络协议、Java 包名使用稳定英文内部 ID。
- 玩家可见文本集中在 `lang/zh_cn.json` 和 `lang/en_us.json`。
- 途径、序列、扮演、魔药、仪式、封印物和路线数据保留 JSON 驱动入口。
- `/reload` 可更新纯数据内容；需要执行器的行为由稳定 ID 映射到 Java 实现。

## 4. 晋升与失控规则

### 低序列晋升

1. 当前途径必须与目标魔药一致。
2. 当前序列必须恰好比目标序列低一级。
3. 当前魔药消化达到 100%。
4. 在坩埚中按正确顺序、材料与火候完成目标魔药。
5. 服药时结算品质、初始压力和污染。

### 失控模式

| 模式 | 后果 |
|---|---|
| recoverable | 倒地、生成途径失控体、掉落破碎特性、获得精神创伤 |
| permanent | 序列降级或身份破碎、知识封锁、掉落破碎特性 |
| death | 死亡并凝聚完整特性 |

失控体基础特性掉率为 60%。占卜家失控体“疯言先知”会泄露附近玩家坐标并预判闪避。

## 5. 扮演法

- 每个序列拥有一句原创扮演准则。
- 扮演事件提供基础消化值，并受新颖度、风险、魔药品质和服务器倍率修正。
- 对同一目标或同一行为的重复触发会衰减。
- 违反准则可以直接增加压力或扣除消化。

## 6. 占卜家途径 M1 纵切

定位：信息与准备优先，正面数值弱于同序列战斗途径，通过预判、欺骗和退路获得优势。

### 序列 9 · 占卜家

| 项目 | 数值 |
|---|---:|
| 灵性上限 | 122 |
| 服药压力 | +10 |
| 主材料 | 夜蚀花种子（当前工程以灵性草药和占卜水晶组成可玩配方） |

能力：

- `V` 灵视：观察灵体光环、情绪色和污染线索。
- 危险直觉：致命攻击前获得一次规避机会。
- `B` 简易占卜：获得带可信度分级的是/否线索。

准则：不把占卜当作答案，不窥探自身与不可承受的存在。

### 序列 8 · 小丑

| 项目 | 数值 |
|---|---:|
| 灵性上限 | 144 |
| 服药压力 | +12 |
| 主材料 | 深灰灵体之泪 |
| 辅材 | 石楠、灵性酒精 |
| 火候 | 文火，工程理想温度 65 |

能力：

- `R` 纸牌飞刃：消耗 3 张秘术纸牌与 12 灵性，三连发合并结算；携带火焰弹可附灵性燃烧。
- 完美平衡：降低摔落伤害和击退。
- 潜行右键村民：表情操控，消耗 8 灵性并短暂改善交易条件。
- 直觉预判：15% 概率完全闪避近战，冷却 8 秒。

扮演事件：

- 在至少两名见证者面前完成纸牌表演。
- 压力达到 50 时仍完成战斗。
- 累计完成五次直觉闪避。

准则：以笑容掌控痛苦，以平衡驾驭危险。

### 序列 7 · 魔术师

| 项目 | 数值 |
|---|---:|
| 灵性上限 | 175 |
| 服药压力 | +15 |
| 主材料 | 幻形蛇毒腺 |
| 辅材 | 灰烬粉、银屑 |
| 火候 | 武火，工程理想温度 90 |

能力：

- `Z` 火焰跳跃：消耗 18 灵性，最多闪现 8 格并点燃落点附近敌人。
- `X` 替身纸人：消耗纸人和 25 灵性，30 秒内替玩家承受一次致命攻击。
- `C` 空气子弹：消耗 6 灵性，中距离伤害与轻微击退。
- `M` 舞台幻术：消耗 20 灵性，诱导附近生物在 10 秒内走向虚假目标。

扮演事件：

- 用替身纸人逃脱致命攻击。
- 火焰跳跃后 5 秒内用空气子弹命中。
- 一次舞台幻术误导至少 3 个生物。

准则：准备真正的退路，再让所有人只看见奇迹。

## 7. M1 基础内容

### 物品

- 坩埚、调查员残缺手稿、灵性草药、占卜水晶、月华水。
- 调查员罗盘、灵性盐、镇静熏香、灵视提灯、灰烬丝线。
- 深灰灵体之泪、石楠、灵性酒精。
- 幻形蛇毒腺、灰烬粉、银屑。
- 秘术纸牌、替身纸人。
- 占卜家、小丑、魔术师三种魔药。

### 生物

- **疯言先知**：占卜家失控体，泄露坐标、施加黑暗与虚弱，并概率预判攻击。
- **幻形蛇**：在主世界低概率生成，接近前保持隐匿，掉落魔术师魔药主材料。
- **灵体微光**：低权重飞行灵体，周期性抽取灵性，掉落灵性盐与概率深灰灵体之泪。
- **灰烬傀儡**：低速高耐久亡灵，攻击施加虚弱与缓慢，掉落灰烬丝线与概率灰烬粉。

### 玩家引导

- 首次登录获得《调查员残缺手稿》和调查员罗盘。
- 右键手稿根据途径、序列和消化进度显示下一步。
- 右键罗盘或 `/pm camp` 指向最近营地；尚无记录时指向世界种子确定的新手营地。
- `N` 打开非凡者档案，显示灵性、消化、污染、压力和知识。
- `/pm m1check` 按序列与消化状态给出下一道门槛；规则、物品和生物均有独立查询命令。
- 调查营地战利品提供手稿、罗盘、基础材料和短期风险管理物品。
- 八段教程成就覆盖从获得手稿到晋升魔术师的完整纵切。

详细流程见 [`GETTING_STARTED.md`](GETTING_STARTED.md)。

## 10. 原创化与发布门槛

- 禁止提交小说原文、官方插画、官方 Logo、提取模型、未授权音乐或字体。
- 术语映射见 [`IP_MAPPING.md`](IP_MAPPING.md) 和 [`ip_mapping.csv`](ip_mapping.csv)。
- 原创代码使用 GPL-3.0-or-later；原创文档和美术使用 CC BY-NC-SA 4.0。
- 第三方名称、角色、设定、商标和素材不因仓库许可证获得再授权。

## 11. 测试与验收

- 序列间 DPS、生存和功能差异需保持可解释。
- 消化目标为每序列段 3–4 小时活跃玩法，挂机收益为 0。
- 死亡、重登、跨维度和服务端重启必须保留玩家状态。
- 网络能力必须由服务端重新验证目标、距离、资源、冷却和 PvP 权限。
- Pages、Wiki、README、CHANGELOG 和路线状态必须与代码同步。

## 12. 当前执行结论

当前仓库已完成占卜家序列 9–7 的代码纵切、确定性新手入口、野外风险工具、三类自然生物、
八段教程成就、能力、扮演事件和可查询引导。M1 进入验收 Beta，仍需完成真实的一小时生存、
死亡重连、服务端重启与平衡记录；验收通过前不得推进当前里程碑到 M2。
