## 决策说明

### 内容 schema v4 不等于玩家 schema 4

v0.9 的 schema v4 描述的是数据内容定义。仓库玩家 Capability 已经使用 schema 15，因此不能降级为 4；当前按向前兼容原则升至 schema 17。

### 现有玩法作为迁移资产

五途径序列 9–7、三委托、三任务链、轻量雾都与持久队伍功能均保留并继续可玩，并按 v0.9 新门禁重新归档。M0/M1 已满足机器可验证退出条件；M2 及后续仍必须按各自合同独立验收。

### 原作概念与实现边界

仓库继续使用非官方同人定位。代码采用 GPL-3.0-or-later，原创文档与美术采用 CC BY-NC-SA 4.0；原作名称、设定、商标和第三方素材不因仓库许可而被重新授权。新内容应优先采用 `docs/IP_MAPPING.md` 中的原创化映射，并通过 schema v4 的 `canon_status`、`source_tier` 与 `source_refs` 标记来源策略。
