# M2 神秘学家小屋、真假配方与三解救援

> 适用版本：0.9.5-1.20.1
>
> 路线定位：本页内容是 v0.9 M2 调查与生活阶段的可玩迁移资产，当前正式里程碑仍是 M0

## 如何进入

1. 完成“走失的煤球”和“失踪的调查小队·第一阶段”。
2. 在雾都前哨右键委托板，或执行 `/pm commission accept counterfeit_formula`。
3. 使用 `/pm case` 定位神秘学家小屋；小屋距离出生区约 8–10 区块，按世界种子确定。
4. 与神秘学鉴定师莫尔交谈，领取密封配方卷宗。

委托数据中的 `prerequisites` 会阻止玩家跳过失踪小队案件。小屋、NPC 和卷宗都具备旧世界/遗失恢复路径，
不要求重新开图或使用管理员命令。

## 真假配方鉴定

卷宗包含隐藏的服务器权威真伪结果和三项可见证据。右键卷宗或执行 `/pm formula inspect`：

| 鉴定路线 | 条件 | 消耗 |
|---|---|---:|
| 灵性检查 | 占卜家途径，序列 9 或更高 | 6 灵性 |
| 试剂分析 | 同时携带占卜水晶和神秘墨水 | 1 神秘墨水；水晶不消耗 |

鉴定后会显示水印、墨迹和材料顺序三项线索：

- 三项全部一致：执行 `/pm formula verdict authentic`。
- 任意一项异常：执行 `/pm formula verdict forged`。
- 错误判断不会永久卡步，但会增加 8 点失控压力。
- 正确判断解锁“真假配方三证鉴定”调查准则，再把卷宗交还莫尔结算 72 便士与 5 点事务所声望。

## 失踪记者三解

抵达邪教救援营地并进入记者营救步骤后，必须选择一次路线：

```text
/pm commission approach assault
/pm commission approach stealth
/pm commission approach divination
```

| 路线 | 门槛 | 玩法与代价 |
|---|---|---|
| 强攻 | 无途径限制 | 生成四名带队伍隔离标记的仪式守卫；全部击败后才能营救 |
| 潜入 | 偷盗者或学徒序列 9 或更高 | 在线队员获得限时隐形和加速，绕过正面守卫战 |
| 占卜 | 占卜家序列 9 或更高 | 消耗 12 灵性，找到安全接近记者的路线 |

选择在当前委托中锁定，放弃委托才会重置。强攻守卫按玩家/原版记分板队伍隔离，不会被其他调查队的击杀清空；
潜入与占卜的准备状态会同步给同一任务链中 2–4 名在线队员。

## 持久化与恢复

- Capability schema 17 保存 `questResolutionRoute` 和 `questResolutionReady`。
- 死亡、重登、跨维度和服务器保存后保留路线；结算或放弃时完整清理。
- 记者或鉴定师实体被命令清除后，只要对应结构区块重新加载，服务器会自动补回任务 NPC。
- 卷宗丢失时再次右键莫尔即可获得同一案件的新卷宗；其真伪仍由世界、玩家与接取时间确定。
- 已完成鉴定但步骤尚未写回时，再次右键卷宗会恢复任务推进，不重复消耗灵性或神秘墨水。

## 自动门禁

```bash
python scripts/gen_datapack.py --check
python scripts/check_m2_investigation.py
./gradlew test
python scripts/run_server_smoke.py --timeout 180
```

`docs/m2-investigation-contract.json` 冻结以下边界：

- 3 个委托与 3 条任务链。
- 失踪小队旧 13 步数量和前七步兼容前缀。
- `assault` / `stealth` / `divination` 三解名称。
- 真假配方前置、六步顺序、命令入口、双语键与 Capability schema 17。

## 当前边界

小屋和三解分支是可运行的轻量预研，不代表完整雾都镇区、正式潜入 AI、警戒等级、剧情镜头或 6–10 小时精制调查链已经完成。

2–4 人队伍的步骤、路线、记者和夜袭状态会写入持久调查账本。已登记成员离线后上线自动追赶，
中途加入使用 `/pm party sync`；个人仍分别结算奖励。详见
[`M2_PERSISTENT_PARTY_RECOVERY.md`](M2_PERSISTENT_PARTY_RECOVERY.md)。
M0/M1 的机器门禁已经关闭并正式推进到 M2；真人两小时与多人手感记录继续作为发布质量回归。
