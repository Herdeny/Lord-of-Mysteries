    About

    关于本项目

    项目简介

    Project Mystery 是一个把《诡秘之主》非凡者体系带入 Minecraft 的生存冒险 Mod。当前按 v0.9 路线处于 M0 内容基建 / M1 纵切迁移：schema v4 公共元数据、71 节点/63 关系内容图、Capability schema 16、正式 DataFix、迁移前世界快照、未知数据孤儿保留、分层非凡特性、扮演 v2、网络协议 7 和完整 CI 门禁已经落地。五条序列 9–7、三种委托、三条任务链、轻量雾都和 2–4 人持久调查作为可玩迁移资产保留；100 节点门禁、dirty mask、完整生命周期 GameTest、特性提取、两小时教程、正式城市与动态案件仍待完成。
