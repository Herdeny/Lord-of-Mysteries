第一次游玩

  1. 安装 Forge 1.20.1 与本 Mod
  2. 进入世界后输入 /pm next
  3. 跟随罗盘抵达调查员营地
打开入门指南 ↗

继续 M2 调查

  1. 完成占卜家序列 9–7 纵切
  2. 输入 /pm city 查看三类服务台
  3. 使用事务所、巡警、证据与报纸
打开城市服务手册 ↗

    About

    关于本项目

    项目简介

    Project Mystery 是一个把《诡秘之主》非凡者体系带入 Minecraft 的生存冒险 Mod。当前版本为 0.9.6-1.20.1，路线处于 M2 调查与生活：M0/M1 已完成机器验收，服务端权威调查板包含案件公告和三案证据档案；日报、报社轮班、事务所补给和巡警安全室形成首批城市服务循环。当前基线包含 Capability schema 17、网络协议 10、218 项 JUnit、6 项 Forge GameTest、精确回滚与双启动专服矩阵；下一阶段继续建设正式城市街区、证据关联推理、动态案件和长期多人经济。
