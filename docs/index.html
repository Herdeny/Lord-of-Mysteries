    关于本项目

    项目简介

    Project Mystery 是一个把《诡秘之主》非凡者体系带入 Minecraft 的生存冒险 Mod。当前为 M1 功能闭环 Alpha：占卜家序列 9、坩埚炼药、扮演消化、净化封印、专属失控体与废弃调查营地基础均已可玩。
