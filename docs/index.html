Start Here

从你的目标开始

不用先读完设计文档。选择身份，按最短路径进入游戏、调查或开发。

第一次游玩

  1. 安装 Forge 1.20.1 与本 Mod
  2. 进入世界后输入 /pm next
  3. 跟随罗盘抵达调查员营地
打开入门指南 ↗

M2 调查与城市生活

  1. 完成占卜家序列 9–7 纵切
  2. 输入 /pm city 查看三座正式街区；用 /pm life 查看工作、事件和暴露
  3. 在调查板切换案件、证据与关联推理
  4. 使用 /pm case analyze 获取明确下一步
  5. 使用 /pm case hypothesis 提出、检验并纠正玩家假说
  6. 丢失关键案件物品时在调查板附近执行 /pm case recover
  7. 完成真假配方后接取轮换案件，用 /pm case rotation 查看八槽位卷宗
  8. 在现场辨认案件相关人、受影响者与可视物证；用证物袋封存差异化样本，再右键指定证人和当日报纸
  9. investigate field|desk 保留为兼容/恢复入口；误判后回调查板复议
  10. 结案后领取组织响应，与对应 NPC 简报，再回调查板提交；rotation contacts 可查看联系人和组织立场
打开动态案件手册 ↗

M3 序列 6–5

  1. 五条首发途径均可从序列 7 继续炼制 6、5 魔药
  2. Y / U 使用每序列主、副能力
  3. 当前状态为 code_ready：专属仪式与完整原典系统仍在建设

    About

    关于本项目

    项目简介

    Project Mystery 是受《诡秘之主》启发的 Minecraft 生存冒险 Mod。当前版本为 0.9.18-1.20.1，正式里程碑仍是 M2 调查与生活：三座 7×7 正式街区、4 种委托、4 条任务链、三类八槽位动态案件、6 类主体关系、12 项周指令、五档组织立场、跨案件联系人记忆、三职业经济、神秘暴露和 16 日六事件周期已经连接为服务端权威闭环。M2 只保留四人八小时真人专服与经济平衡证据，不能由自动化替代。M3 已预实现五条首发途径序列 6–5：10 瓶魔药、10 份 schema v4 序列定义、20 项服务端能力和 20 项扮演事件；玩家控制默认禁止，PvP、Boss、物品归属、书本代数、每日窃梦和安全传送均有边界。当前基线为 Capability schema 23、网络协议 12、340 项 JUnit、9 项 Forge GameTest、126 节点/188 关系内容图、精确回滚和双启动专服矩阵。图鉴注册内容由脚本从 Java 注册表和双语文件生成，不再依赖手工补条目。
