# M2 系统成型计划

技术基线继续以仓库现有 Forge 1.20.1 / Java 17 实现为准。当前设计基线为 v0.6；
设计文档中的其他加载器或版本代码只用于机制和数值参考。

## M2-1 观众序列 9–8

状态：已实现，等待多人游戏验收。

- 观众魔药：灵性草药 → 发酵蛛眼 → 可选蜂蜜瓶。
- 读心者魔药：灵性草药 → 书 → 可选紫水晶碎片。
- 序列 8 晋升要求：同途径序列 9 且消化达到 100%。
- 情绪观察：持续消耗 0.5 灵性/秒，只读取游戏内推导的情绪状态。
- 行为预测：8 秒冷却，降低一次来袭伤害 40%。
- 表层读心：只返回仇恨、恐惧、好奇或平静等游戏状态，不读取聊天或背包。
- 心理暗示：服务端验证目标、距离、灵性、冷却和 PvP 配置；目标玩家获得明确提示，
  潜行时持续时间从 8 秒缩短为 3 秒。
- 玩家数据 schema 升级至 2，新增能力开关、冷却和扮演计数持久化。

## M2-2 猎人序列 9–8

状态：已实现，等待多人游戏验收。

- 猎人魔药：灵性草药 → 骨头 → 可选兔子脚。
- 挑衅者魔药：灵性草药 → 火药 → 可选红石。
- 序列 8 晋升要求：猎人序列 9 且消化达到 100%。
- 追踪：攻击目标后保留 45 秒追踪标记；追踪超过 30 秒后击杀可推进扮演。
- 陷阱精通：猎人捕兽夹绑定放置者，触发后施加缓慢与虚弱并自动消耗。
- 荒野感知：在露天环境提示 24 格内最近的敌对或战斗目标。
- 挑衅：`K`，15 灵性，20 秒冷却，使 10 格内低抗性敌人优先攻击玩家 8 秒。
- 激怒：`L`，20 灵性，30 秒冷却，使目标攻击提高 30%、护甲降低 25%，持续 8 秒。
- 战斗意志：至少 3 名敌人锁定玩家时获得 15% 减伤 10 秒，45 秒冷却。
- 玩家数据 schema 和网络协议升级至 3；目标、冷却和持续时间均由服务端校验。

## M2-3 通用仪式底座

状态：已实现首个完整仪式，等待多人游戏验收。

- 新增仪式粉笔阵纹：骨粉 + 木炭一次制作 8 个。
- 祭坛半径 3 的八点阵列偏移为 `(±3,0)`、`(0,±3)` 与 `(±2,±2)`。
- 启动时校验阵列、材料和环境；进行期间每 20 tick 复检结构。
- 阵列破坏会进入 `CANCELLED`，不消耗纯净水和青兰花。
- 结算分数由材料、环境、结构完整度和主持者资格组成，并叠加小幅服务端随机扰动。
- 结果分为完美、成功、失败、严重失败和灾变；失败会增加压力或污染，
  高危结果还会生成失控体，灾变附带非破坏性爆炸。
- 成功封印写入 `sealed` 与 `seal_quality` NBT；可消耗材料在所有正常结算后统一扣除。
- 当前只接入净化封印；尊名呼名、多人参与加成、晋升饮药窗口和其他仪式留待后续批次。

## 验收门槛

- `./gradlew clean build` 通过。
- 普通人、错误途径、未完成消化均无法绕过晋升门槛。
- 客户端不能直接修改灵性、冷却、目标或效果持续时间。
- 对玩家的精神能力有服务端开关、可见提示和主动抵抗。
- 新增资源 ID 使用稳定英文代号，玩家可见名称保留原创化替换能力。
