# Project Mystery 入门指南

> 适用版本：0.3.0-1.20.1  
> 当前路线：v0.6 M1 占卜家序列 9–7 纵切

## 1. 第一次进入世界

首次登录会获得《调查员残缺手稿》。右键它可以查看当前目标；按 `N` 打开非凡者档案。

推荐流程：

1. 探索调查员营地，收集手稿、灵性草药和占卜水晶。
2. 制作坩埚并在下方放置热源。
3. 按顺序放入材料，空手右键坩埚开始炼制。
4. 观察剩余时间和温度，完成后取出魔药。
5. 服药后通过对应扮演行为消化，达到 100% 才能晋升。

## 2. 占卜家魔药

- 核心材料：灵性草药、占卜水晶。
- 可选材料：月华水，提高品质。
- 理想温度：70。

序列 9 能力：

- `V`：灵视。
- `B`：简易占卜。
- `N`：状态与知识。

## 3. 小丑魔药

- 主材料：深灰灵体之泪。
- 辅材：石楠、灵性酒精。
- 理想温度：65。
- 前置：占卜家魔药消化 100%。

序列 8 能力：

- `R`：纸牌飞刃，需要 3 张秘术纸牌；携带火焰弹时附加燃烧。
- 被动：摔落与击退降低，近战有 15% 概率直觉闪避。
- 潜行右键村民：表情操控，短暂改善交易。

秘术纸牌由纸和墨囊制作；替身纸人由纸和线制作。

## 4. 魔术师魔药

- 主材料：幻形蛇毒腺。
- 辅材：灰烬粉、银屑。
- 理想温度：90。
- 前置：小丑魔药消化 100%。

幻形蛇会在主世界低概率生成，接近玩家前保持隐匿。

序列 7 能力：

- `Z`：火焰跳跃，最多 8 格。
- `X`：布置替身纸人，30 秒内规避一次致命攻击。
- `C`：空气子弹。
- `M`：舞台幻术，误导附近敌对生物。

## 5. 扮演与风险

- 同一行为反复执行会降低消化收益。
- 魔药品质会影响消化效率、初始污染和压力。
- 污染与压力过高会触发幻觉、负面状态或失控。
- 占卜家失控体会公开附近玩家坐标，并概率闪避攻击。

## 6. M1 验收玩法

一小时纵切应覆盖：

1. 获取或制作基础材料。
2. 完成序列 9 魔药与至少一次占卜/灵视使用。
3. 验证扮演、消化、死亡和重登持久化。
4. 使用小丑纸牌、闪避和表情操控。
5. 使用魔术师火焰跳跃、替身、空气子弹和舞台幻术。
6. 遭遇幻形蛇和占卜家失控体。

当前代码纵切已完成；正式推进 M2 前仍需记录一小时生存回归和数值调整结果。
