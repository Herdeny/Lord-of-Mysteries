# M1 执行计划 — 占卜家序列 9-7 纵切

> 设计基线：Project Mystery v0.8
>
> 工程基线：Minecraft 1.20.1 · Forge 47.4.20 · Java 17
> 路线来源：`roadmap.json`

## 验收门槛

M1 不是“占卜家序列 9 已可玩”，而是普通玩家能在生存模式连续完成
“普通人 → 序列 9 占卜家 → 序列 8 小丑 → 序列 7 魔术师”的一小时纵切。

- 三个序列均具备魔药、晋升门槛、核心能力、扮演事件、污染和失控反馈。
- 客户端只能发起意图，目标、消耗、冷却和结果由服务端裁决。
- 一小时切片覆盖调查入口、材料收集、炼药、晋升、战斗和消化。
- 序列 9-7 的 DPS、生存和功能差异可解释，同序列差异控制在目标范围内。
- `./gradlew clean build`、174 项测试、M1/M2 合同、专服运行诊断—命令循环—强制保存—停服冒烟、文档同步和 GitHub Actions 全部通过。

## M1-1 序列 8 小丑（已完成）

- 灵性上限 144，服药压力 +12。
- 纸牌飞刃：4 灵性/张，0.5 秒间隔，支持三连发。
- 完美平衡：免疫滑倒，击退降低 50%，支持窄面移动判定。
- 表情操控：8 灵性，10 秒冷却，作用于游戏内 NPC 欺骗和交易。
- 直觉预判：8 秒冷却，15% 近战完全闪避。
- 扮演：公开表演、压力战斗中保持笑容、单场多次闪避。

## M1-2 序列 7 魔术师（已完成）

- 灵性上限 175，服药压力 +15。
- 火焰跳跃：18 灵性，6 秒冷却，8 格位移并产生落点火焰效果。
- 替身纸人：25 灵性，45 秒冷却，预置后在 30 秒窗口内替代致命伤。
- 空气子弹：6 灵性，1 秒冷却，中距离骚扰。
- 舞台幻术：20 灵性，30 秒冷却，5×5 区域投影并影响敌对 AI 路径。
- 扮演：替身逃脱、华丽入场、诱导多个敌对生物走错路。
