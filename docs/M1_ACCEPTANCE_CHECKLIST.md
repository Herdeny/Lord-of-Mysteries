# M1 两小时纵切验收清单

> 适用版本：`0.9.17-1.20.1` · 自动化实施门禁已完成

## 开始前

- [ ] 新建普通人生存角色，执行 `/pm doctor`。
- [ ] 执行 `/pm trial reset` 与 `/pm trial start`。
- [ ] 记录版本、commit、世界种子、难度、服务端配置与开始时间。
- [ ] 不使用创造模式、管理员物品或外部 NBT 修改推进玩法。

## 0–30 分钟：营地与序列 9

- [ ] 调查手册和罗盘正常发放，`/pm recover` 能补回缺失物品。
- [ ] 罗盘和 `/pm camp` 指向同一个确定性营地。
- [ ] 营地保证提供序列 9/8 所需基础补给，但不直接提供序列 7 毒腺。
- [ ] 按温度和投料顺序炼制、饮下占卜家魔药。
- [ ] 首次服药发放扮演身份卡和反思日志。

## 30–90 分钟：消化、狩猎与序列 7

- [ ] 正收益扮演提高消化，机械重复出现收益衰减。
- [ ] 序列 9 未完全消化时不能晋升序列 8；序列 8 同理。
- [ ] 灵视提灯能显形幻形蛇，毒腺有稳定可再生获取路径。
- [ ] 小丑与魔术师能力由服务端校验灵性、距离、目标和冷却。
- [ ] 三次神秘生物击杀、两次有效扮演和风险峰值被追踪器记录。

## 90–120 分钟：身份、反思与生活

- [ ] 身份卡只能使用预设档案，不要求玩家输入真实个人信息。
- [ ] 右键身份卡锚定当前途径/序列；潜行右键切换四个预设。
- [ ] 反思日志和 `/pm reflect` 共用每日一次冷却，不会重复结算。
- [ ] `/pm mistcity` 能定位前哨；床铺、制作设施、生活桶、街道与 NPC 可用。
- [ ] 携带 3 张纸潜行交互报社职员，获得 24 便士和 2 个面包；每日只能完成一次。
- [ ] `/pm life`、`/pm next` 与 `/pm m1check` 显示一致的生活和完成状态。

## 特性与失败路径

- [ ] 永久失控清除当前途径身份，并把精确分层特性转移到失控体。
- [ ] 击杀失控体只掉落一份携带完整 NBT 的破碎特性，不生成通用碎片复制品。
- [ ] 死亡重生、退出重登、跨维度和专服重启后身份、反思、轮班、余额与试炼状态一致。
- [ ] 恢复脚本 dry-run 不修改世界；实际恢复先创建 `pre-restore-*` 安全副本。

## 结束标准

- [ ] `/pm trial status` 显示核心目标 9/9。
- [ ] `/pm trial report` 显示七个里程碑，超时项有明确记录。
- [ ] `/pm trial verify` 显示连续性证据 4/4。
- [ ] 308 项 JUnit、6 项 Forge GameTest、回滚检查、clean build 与专服双启动矩阵通过。
- [ ] 无崩溃、复制、永久卡步、跨玩家串档或客户端越权。

自动门禁完成后，仍应记录材料耗时、平均战斗时长、灵性净变化、压力/污染峰值、死亡次数、可读性和无障碍反馈，作为后续平衡与发布质量证据。
